Text, Images, and Videos

< b o d y > and < / b o d y > are tags used to put content in a website. For example images or text.
using < p > and < / p > are tags used to create paragraphs
< b r / > is a self-closing tag that does a line break when inserted in code.
You need to indent code properly so that it is easy to read
The < ! - - and - - > are tags that allow you to comment to allow the reader to understand what the code means
< d i v >s can contain any text or other HTML elements. They are primarily used to divide HTML documents into sections.
< s p a n >s contain short pieces of text or other HTML. They are primarily used to wrap small pieces of content that are on the same line as other content and do not break text into different sections.

Links

Links can be inserted in a website with the < a > tag. You also need a href element between the tags to insert the url..
You can add links to images by wrapping the image tag with < a > tags.
The target allows you to make the link create a new browser window.
You can create sections on a webpage and jump to them using < a > tags and addings ids to the elements you wish to jump to.
The nav element contains links to internal pages or content.

Tables

The < t a b l e > allows you to create tables.
< tr > allows you to put rows in your tables.
< t d > is what allows you to insert content inside the cells in a row
< t h > is a tag used to put headings in tables.
You can use the colspan attribute to make the cell in a table longer
You can use the rowspan attribute to make a cell in a table deeper
You can split the table in three parts: thead(headings),tbody(table data), and tfoot(last row)
