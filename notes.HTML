< h 1 > is usually the main header and < h 2 >, < h 3 >, < h 4 >, etc. are used for sub-sections.
< h e a d > and < / h e a d> are tags for the metadata of a website. Metadata is information about the page that isn't displayed directly on the web page.

< t i t l e > and < / t i t l e > are used to create the title of the webpage. It is what you see on the tab of a webpage. It goes in the head tags.
Anything between the opening < h t m l > and closing < / h t m l > tags will be interpreted as HTML code
< b o d y > and < / b o d y > are tags used to put content in a website. For example images or text.
using < p > and < / p > are tags used to create paragraphs
< b r / > is a self-closing tag that does a line break when inserted in code.
You need to indent code properly so that it is easy to read
The < ! - - and - - > are tags that allow you to comment to allow the reader to understand what the code means
< d i v >s can contain any text or other HTML elements. They are primarily used to divide HTML documents into sections.
< s p a n >s contain short pieces of text or other HTML. They are primarily used to wrap small pieces of content that are on the same line as other content and do not break text into different sections.
